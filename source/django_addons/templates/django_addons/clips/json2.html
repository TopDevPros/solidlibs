Sometimes when our code is wrong it helps to step through json stringify() or parse(). In both cases we use the reference implmentation of json, "json2". It will not replace existing JSON, so we remove the old JSON first with "JSON = {};". https://github.com/douglascrockford/JSON-js/blob/master/json2.js {% endcomment %}
